# Requisitos do Sistema

## Requisitos Funcionais (RF)

### RF01 — Login do usuário
Permitir que o usuário realize acesso ao sistema utilizando e-mail e senha previamente cadastrados.

### RF02 — Recuperar senha
O sistema deve permitir a solicitação de recuperação de acesso através de um código de verificação enviado por e-mail.

### RF03 — Cadastro de Usuário
O sistema deve permitir a criação de novas contas mediante fornecimento de nome, e-mail, senha e aceite dos termos de uso.

### RF04 — Registro de Descarte por Imagem
O sistema deve capturar a foto do resíduo descartado para fins de validação e início do processo de pontuação.

### RF05 — Feedback Educativo
Após o registro, o sistema deve exibir informações técnicas sobre o material (ex: Plástico BOPP) e dicas de reciclagem.

### RF06 — Atribuição de Eco Pontos
O sistema deve creditar automaticamente os pontos correspondentes na conta do usuário após a confirmação do descarte.

### RF07 — Dashboard de Pontuação
A tela inicial deve exibir de forma clara o saldo total de "Eco Pontos" acumulados pelo usuário.

### RF08 — Listagem de Parceiros
O sistema deve listar as barracas e estabelecimentos parceiros da universidade, exibindo os descontos oferecidos por cada um.

### RF09 — Resgate de Cupom
Funcionalidade que permite ao usuário trocar seus pontos acumulados pelos benefícios escolhidos.

### RF10 — Geração de Código
Emissão de um código alfanumérico único ou QR Code para que o desconto seja validado presencialmente na barraca.

### RF11 — Ranking de Usuários
Exibição de um Leaderboard (quadro de líderes) para comparação de pontuações entre os usuários da universidade.

### RF12 — Conteúdo Informativo
Seção dedicada à educação ambiental, contendo textos e guias didáticos sobre a importância do descarte correto.

### RF13 — Chatbot de Suporte (IA)
Interface de chat automatizada para esclarecer dúvidas frequentes sobre o descarte e o uso do aplicativo.

### RF14 — Canal de Suporte Direto
Opção para o usuário enviar mensagens personalizadas e anexar imagens para a equipe de suporte técnico em caso de erros.

### RF15 — Edição de Perfil
O usuário deve ser capaz de alterar sua foto de perfil, nome de exibição e outros dados pessoais.

### RF16 — Alteração de Senha Interna
O sistema deve permitir que o usuário logado altere sua senha de acesso nas configurações, mediante confirmação da nova senha.
